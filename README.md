# EduQuiz

EduQuiz is a desktop educational quiz application built with Electron, TypeScript, and Vite.

## Features

- **Create Quizzes**: Build custom quizzes with questions, multiple choice answers, and images
- **Take Quizzes**: Load and complete quizzes with timer support
- **Check Results**: Review quiz results, analyze performance, and export to CSV
- **Cross-platform**: Works on Windows, macOS, and Linux

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```
   This will start the Vite dev server and launch Electron.

3. Build for production:
   ```bash
   npm run build
   ```

4. Package the app:
   ```bash
   npm run electron:pack
   ```
